# Grade Point Calculator

A responsive web-based GPA calculator for Raghu Engineering College (REC) and Raghu Institute of Technology (RIT).

## Features

-  Separate calculators for REC and RIT with different grading scales
-  Fully responsive design
-  Modern dark theme with glassmorphism effects
-  Zero dependencies - pure HTML, CSS, and JavaScript
-  Real-time GPA calculation

## Grading Scales

### REC (Raghu Engineering College)
- O = 10 points
- S = 9 points
- A = 8 points
- B = 7 points
- C = 6 points
- F = 0 points (Fail)

### RIT (Raghu Institute of Technology)
- A+ = 10 points
- A = 9 points
- B = 8 points
- C = 7 points
- D = 6 points
- F = 0 points (Fail)

## How to Use

1. Open `index.html` in your browser
2. Select your college (REC or RIT)
3. Add subjects and select grades and credits
4. Click "Calculate GPA" to see your results

## Files

- `index.html` - Landing page with college selection
- `rec-calculator.html` - REC calculator
- `rit-calculator.html` - RIT calculator
- `script-rec.js` - JavaScript for REC calculator
- `script-rit.js` - JavaScript for RIT calculator
- `style.css` - Shared styles
